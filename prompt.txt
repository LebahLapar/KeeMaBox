Fondasi Phase 1 sudah berjalan. Mari kita eksekusi kode untuk Phase 2: Backend Development & Cloud Integration.

Tolong modifikasi secara menyeluruh file app.py dengan logika berikut:

    Database: Setup pymongo untuk menunjuk ke database smart_box dan collection packages.

    MQTT: Integrasikan paho-mqtt untuk terhubung ke broker.avisha.id port 1883 dengan username barka (sediakan placeholder konfigurasi password). Buat agar client MQTT langsung connect saat aplikasi Flask berjalan.

    Telegram: Buat fungsi send_telegram_notification(resi, barang) menggunakan library requests. Gunakan os.environ.get untuk mengambil TELEGRAM_BOT_TOKEN dan TELEGRAM_CHAT_ID. Pesannya berisi info paket tiba dan meminta pemilik membuka dashboard.

    Rute API Validasi (/api/validasi_resi - POST): Menerima JSON { "resi_number": "..." }. Lakukan pengecekan ke MongoDB. Jika resi ada, panggil fungsi notifikasi Telegram dan kembalikan response sukses.

    Rute API Aktuator (/api/buka_pintu - POST): Saat dipanggil, lakukan mqtt_client.publish payload "OPEN" ke topik barka/kontrol.

    Rute API Kamera (/api/upload_frame - POST): Buat kerangka dasar (dummy) yang menerima file gambar dan mengembalikan response "Frame diterima" (sebagai persiapan ESP32-CAM nanti).

Tolong berikan kode app.py versi penuh (full code) yang bersih dan siap dijalankan.
